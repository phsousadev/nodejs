# Introdução ao Node.js

Antes de tudo, vale reforçar que **Node.js não é uma linguagem de programação** como PHP, Java ou Python. Ele é, na verdade, uma **plataforma** que permite rodar JavaScript (ou TypeScript) **fora do navegador** — algo que não era comum antes de sua criação.

Por muito tempo, o JavaScript foi usado apenas no front-end, para dar vida às páginas: responder a cliques, alterar elementos na tela e lidar com pequenas lógicas do dia a dia. Com o Node.js, o JavaScript ganhou um novo propósito: passou a rodar também **no servidor**, abrindo espaço para APIs, serviços e aplicações escaláveis, competindo de igual para igual com tecnologias mais consolidadas.

---

## Breve histórico

O Node.js apareceu em **2009**, criado por **Ryan Dahl**. A ideia surgiu quando ele percebeu que, se os navegadores já tinham um motor rápido para interpretar JavaScript, nada impedia que esse motor fosse usado fora deles. Essa sacada deu origem ao Node.js e mudou completamente o papel do JavaScript no desenvolvimento web.

---

## A tecnologia por trás do Node.js

O motor por trás do Node.js é a **V8**, a mesma engine usada pelo Google Chrome. Ela é conhecida por ser extremamente rápida e eficiente, o que ajudou a impulsionar várias tecnologias que surgiram depois ou ganharam força graças ao Node, como:

- React Native  
- Ionic  
- NativeScript  
- Next.js
