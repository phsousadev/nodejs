# Introdução ao Node.js

Primeiramente, é importante esclarecer que **Node.js não é uma linguagem de programação**, como PHP, Java ou Python. Na verdade, trata-se de uma **plataforma** que permite utilizar JavaScript (ou TypeScript) para executar algoritmos **fora dos navegadores** como Chrome, Microsoft Edge e Firefox.

Antes do surgimento do Node.js, o JavaScript nasceu com foco em lidar com a parte funcional de sites e landing pages, sendo responsável por manipular elementos na tela, responder a eventos de clique e realizar pequenas lógicas na interface. Com o surgimento do Node.js, o JavaScript ganhou a capacidade de ser executado **no lado do servidor**, tornando-se possível desenvolver APIs, serviços web e aplicações altamente escaláveis, assim como já era feito com tecnologias como Java e Python.

---

## Breve histórico
O Node.js surgiu em **2009**, criado por **Ryan Dahl**, que teve a ideia de aproveitar o mesmo mecanismo usado pelos navegadores para interpretar JavaScript e utilizá-lo fora deles.

O raciocínio foi simples: se os navegadores já executavam JavaScript para manipular interfaces, por que não reaproveitar o engine responsável por interpretar esse código para outros fins, como processamento no servidor?

---

## A tecnologia por trás do Node.js
Para funcionar, o Node.js utiliza a **V8**, a engine do Google Chrome, um dos motores JavaScript mais rápidos e eficientes do mercado.

Esse avanço possibilitou o surgimento ou a evolução de diversas tecnologias importantes, como:
- React Native
- Ionic
- NativeScript
- Next.js
